Objetivo

Este projeto tem como objetivo criar uma aplicação completa com gráficos no front-end usando Chart.js, e CRUDs completos em todas as camadas e os principais indicadores de faturas emitidas por uma empresa.

Tomei a iniciativa em colocar algumas integrações que achei válidas, como por exemplo o método que emite faturas automaticamente de acordo com a duração do contrato. Desse modo, conseguimos criar toda uma cadeia de boletos de um só vez.

📈 Gráficos e Indicadores

  • 📌 Gráfico de Pizza: Distribuição das faturas por status (Paga, Pendente, Atrasada).
  • 📌 Gráfico de Barras: Evolução mensal do total de faturas emitidas e pagas.
  • 📌 Indicadores Numéricos (Cards):
    • Total de faturas emitidas
    • Valor total faturado
  • 🔔 Notificações de Vencimento

⚙️ Requisitos Técnicos

Back-End (.NET Core 8.0)

  • API REST com .NET Core / C#
  • Boas práticas: SOLID, Clean Code, DDD

Banco de Dados

  • PostgreSQL
  • EF Core In-Memory para testes locais com persistência temporária

Front-End (Angular 19.0)

  • Interface em Angular para gerenciar os dados
  • CRUD completo das entidades
  • Design com Angular Material (responsivo e moderno)
